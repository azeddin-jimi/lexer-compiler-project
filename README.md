#  Lexical Analyzer using both:

Python (PLY) Essential
C (Flex)

This project is a simple lexical analyzer built using Python and PLY.

##  Features
- Recognizes numbers, identifiers
- Supports operators (+, -, *, /)
- Handles keywords (if, while, print)
- String detection

##  Example
Input:
x = 5 + 2

Output:
IDENTIFIER: x
ASSIGN
NUMBER: 5
PLUS
NUMBER: 2

##  Technologies
- Python
- PLY (Python Lex-Yacc)

##  Use Cases
- Compiler design
- Cybersecurity (log analysis, detection)
- Text parsing
